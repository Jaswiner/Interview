HTTP中GET和POST的区别和数据包格式

# 从设计角度来说
- GET请求是幂等的，请求前后不会对数据造成变化
- POST是向服务端提交数据，提交数据就是为了修改相关数据，所以不是幂等的

# 从应用角度来说
- GET的请求有长度限制，URL最大长度为2048byte
- POST请求没有长度限制

# 从浏览器角度来说
- GET请求回退和刷新不会产生危害
- POST会重新提交请求

- GET可以被缓存
- POST不能被缓存
