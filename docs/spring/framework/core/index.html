Spring的加载流程是什么

# 主要类
Spring加载中有以下几个重要的类：
- BeanFactory：生产Bean
- Resource：Bean配置文件
- BeanDefinition：Bean定义

# 流程
- 资源定位：找到配置文件
- BeanDefinition载入和解析
-- 找到配置文件Resource
-- 将配置文件解析成BeanDefinition
- BeanDefinition注册
-- 将BeanDefinition向Map中注册Map<name,BeanDefinition>
- bean的实例化和依赖注入
-- 通过反射或者cglib来创造bean

# Tips:
- Resource本质上就是字节流
- BeanDefinition是bean的详细信息
