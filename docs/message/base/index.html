为什么使用消息队列？优缺点？比较一下Kafka，ActiveMQ，RabbitMQ，RocketMQ
# 为啥使用
- 业务需要
    - 解藕
    - 异步
    - 削峰

消息队列优缺点:

优点 缺点
MQ 1.解藕,异步,削峰 1.系统可用性降低(MQ挂了)
2.复杂度提高(保证消息按照想象的被消费)
3.最终一致性
