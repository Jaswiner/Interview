说说你对Java对象组成的理解

- Java对象内部由三个部分组成:对象头，实例数据，对齐填充字节

# 对象头
- 对象头包括如下部分:
-- MarkWord:记录了对象的锁，GC等等的信息

Garbage Collection(GC)

说说你知道的GC算法和对应的原理是什么？

# 算法
- 标记-清除算法
- 复制算法
- 标记-压缩算法
- 分代收集算法

# 原理
- 先对对象进行标记然后在进行回收(标记和清除效率不高，标记后空间碎片太多)
- 内存一分为二，一边空着另一边放对象，回收时候直接移动存活的对象就好了
- 标记压缩就是多一层压缩
- 对不同年龄代的对象进行回收时候用不同的算法
