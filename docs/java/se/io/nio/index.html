你对NIO是怎么理解的，Java中的NIO是怎么实现的？NIO和AIO,BIO的区别是什么？

# NIO
- 使用缓冲区和通道的概念来实现处理数据
- NIO中通道是双向的
- 缓冲区可以分片

# 实现
- 客户端将流写到缓冲区
- 使用channel将数据从buffer转移

# NIO和AIO,BIO的区别
- AIO是异步非阻塞的，BIO是同步阻塞的，NIO是同步非阻塞的
- AIO也可以叫做NIO2

# Tips
- JDK1.7中NIO中新增了4个异步通道
