深拷贝和浅拷贝的区别

- 对于基本类型来说，浅拷贝和深拷贝一样，都是对值进行拷贝

- 对于引用类型来说:
-- 浅拷贝:只是复制了对应的引用，对象没有被拷贝，并且这个对象的其他引用还在
-- 深拷贝:不只复制了对象的引用，对象本身也被复制了
