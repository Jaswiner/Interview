对象解析

说说你对Java对象组成的理解

- Java对象内部由三个部分组成:对象头，实例数据，对齐填充字节

# 对象头
- 对象头包括如下部分:
-- MarkWord:记录了对象的锁，GC等等的信息
-- 指向类的指针
-- 数组的长度(数组对象才有)

# 实例数据
- 实例数据就是我们在代码中看到的属性和值

# 填充字节
- 对齐填充字节:因为JVM中对象的是8bit的整数倍，所有要有相关的字节来填充
